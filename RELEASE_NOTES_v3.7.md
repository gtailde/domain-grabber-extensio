# 🚀 Domain Grabber v3.7 - Release Notes

**Дата релізу:** 1 лютого 2026

## ⚡ Одноклікове копіювання

### 🎯 Головна новинка: Швидкий workflow

**Проблема:**
- Потрібно було спочатку зібрати домени
- Потім вибрати формат (JSON або Excel)
- І тільки після цього скопіювати

**Рішення:**
- 📥 **Вибір формату перед збиранням** - обираєте JSON або Excel ДО натискання кнопки
- 🎯 **Одна кнопка для всього** - "Зібрати і скопіювати" робить обидві дії одночасно
- ✅ **Автоматичне копіювання** - дані відразу в буфері обміну після збирання
- 📊 **Інформативні нотифікації** - показує скільки доменів знайдено
- 💾 **Запам'ятовування вибору** - формат експорту зберігається між сесіями

## 📋 Формати експорту

### JSON (JSONL формат)
```json
{"domain":"example.com","brand":"DROP","bl":150,"acr":45}
{"domain":"test.net","brand":"DROP","bl":89,"acr":23}
```

**Коли використовувати:**
- Потрібна обробка даних скриптами
- Python, Node.js, або інші мови програмування
- Зручно для автоматизації

### Excel (TSV формат)
```
Brand	Domain	BL	ACR	CF	TF	DP
DROP	example.com	150	45	32	28	15
DROP	test.net	89	23	18	15	8
```

**Коли використовувати:**
- Ручна обробка даних
- Аналіз у Excel або Google Sheets
- Швидкий перегляд результатів

## 💾 Збереження налаштувань

Вибраний формат експорту автоматично зберігається в `localStorage`:
- При наступному відкритті панелі буде обраний той самий формат
- Не потрібно щоразу вибирати JSON або Excel
- Працює між сесіями браузера

---

## 🔧 Технічні деталі

### Зміни в коді:

**1. Додано секцію вибору формату:**
```html
<div class="section-card">
    <h3>📥 Формат експорту:</h3>
    <label id="formatJsonLabel">
        <input type="radio" name="exportFormat" id="formatJson" value="json" checked>
        📋 JSON
    </label>
    <label id="formatExcelLabel">
        <input type="radio" name="exportFormat" id="formatExcel" value="excel">
        📊 Excel
    </label>
</div>
```

**2. Оновлена логіка кнопки:**
```javascript
btnProcess.addEventListener('click', () => {
    const data = processDomains(currentMode);
    const selectedFormat = document.querySelector('input[name="exportFormat"]:checked')?.value;
    
    if (selectedFormat === 'json') {
        const jsonLines = data.map(d => JSON.stringify(d)).join('\n');
        copyToClipboard(jsonLines, `✅ JSON скопійовано!\n\nЗнайдено доменів: ${data.length}`);
    } else {
        // Excel TSV формат
        const tsv = /* формування TSV */;
        copyToClipboard(tsv, `✅ Excel дані скопійовано!\n\nЗнайдено доменів: ${data.length}`);
    }
});
```

**3. Збереження формату:**
```javascript
function saveSettings() {
    const settings = {
        exportFormat: document.querySelector('input[name="exportFormat"]:checked')?.value || 'json',
        // ... інші налаштування
    };
    localStorage.setItem('domain-grabber-settings', JSON.stringify(settings));
}
```
